ЗВІТИ З ЛАБОРАТОРНИХ РОБІТ

з дисципліни «Інтернет-технології та проєктування web-застосувань»

Студентки групи ІС-31
Савенко Анастасії Денисівни

Створення HTML-документів та використання CSS та Flexbox для блокової верстки є ключовими елементами розробки веб-сторінок. Ця лабораторна робота була спрямована на отримання практичних навичок у верстці сторінок та освоєння основних принципів роботи з CSS та Flexbox. Ми досліджували процес створення HTML-документів, їх структуру, а також методи використання CSS для стилізації та оформлення веб-сторінок. Ми також вивчили блокову верстку та інструменти для розташування та позиціонування елементів на веб-сторінці. Важливою частиною нашої роботи було вивчення Flexbox, потужного інструменту для створення адаптивних макетів веб-сторінок. Ми досліджували використання контейнерів Flex та їх властивостей для організації елементів на сторінці, що допомогло нам покращити адаптивність та зручність нашого веб-сайту. Ми також розглянули переваги та недоліки різних типів макетів веб-сторінок, що допомогло нам зробити обґрунтований вибір щодо найкращого підходу до верстки, враховуючи потреби проекту. Ця лабораторна робота дозволила нам поглибити наші знання та навички у верстці веб-сторінок, а також створити адаптивний та ефективний макет за допомогою CSS та Flexbox. Отримані знання та навички дозволять нам успішно використовувати їх у майбутніх веб-проектах.
